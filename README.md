# 🇺🇸 English

## 📖 About

PyMusic is a simple music player developed in Python as a study project and out of curiosity about how audio players work internally.

The project was created with the goal of understanding concepts such as:

- Audio playback
- Graphical interfaces
- State management
- File handling
- Code organization

Instead of using an existing player, the idea was to build one from scratch in a simple and understandable way.

---

## 🎯 Goal

PyMusic is **not intended to compete** with complete music players.

Its purpose is to be:

- Lightweight
- Functional
- Easy to understand
- Easy to expand

The main idea is simply:

- Select music files
- Read local audio files
- Play music
- Control playback
- Listen while doing other activities

---

## 🧩 Features

- 🎵 Local music playback
- ⏯️ Play / Pause controls
- 🔊 Volume control
- 🖼️ Compact graphical interface
- 📂 File selection from the computer
- 🧠 Beginner-friendly structure

---

## 🛠️ Technologies

- Python
- CustomTkinter
- Tkinter
- Pygame
- Pillow (PIL)

---

## 📚 What I learned

This project helped me better understand:

- GUI development with Python
- Audio handling
- Application logic
- State control
- Project organization
- Problem solving during development

---

## 🚀 Future ideas

Possible future improvements:

- Playlist system
- Music progress bar
- Better interface styling
- Keyboard shortcuts
- Drag & drop support
- Multiple audio format support

---

## 📌 Note

This project was made mainly for:

- Learning

- Curiosity
- Practice

PyMusic represents the idea that even simple applications can teach a lot during the development process.
